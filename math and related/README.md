# ∑ Math Stuff

A collection of random mathematical code.

From tiny scripts to slightly more interesting things.
No structure. No roadmap. Just math.

---

## 🧠 What is this?

This folder contains small math-related programs I write when:

* I’m bored
* I want to understand something better
* or I just feel like coding instead of doing something useful

You might find:

* basic algorithms (like factorials)
* small experiments
* weird ideas that probably didn’t need code

---

## ⚠️ Expectations

Don’t expect:

* optimized code
* production-ready logic
* clean architecture

This is more like:

> thinking in code

---

## 🧪 Why this exists

Because math makes more sense when you:

* write it
* break it
* and rebuild it

Also because doing random small things is better than doing nothing.
