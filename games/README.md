# 🎮 Games (kind of)

A collection of small, mostly useless, sometimes funny programs.

Not really games.
Not really serious either.

---

## 🧠 What is this?

This folder contains simple interactive programs that feel like:

* mini games
* logic experiments
* or just ideas turned into code

Some are:

* dumb on purpose
* slightly clever
* or accidentally interesting

---

## 🎯 What to expect

* simple logic
* user input interaction
* random ideas executed quickly
* a mix of serious and not-so-serious concepts

---

## ⚠️ Disclaimer

These are not:

* real games
* polished projects
* or deeply engineered systems

This is more like:

> coding whatever comes to mind

---

## 🧪 Why this exists

Because sometimes:

* you don’t want to build something big
* you just want to build *something*

And small projects are where most ideas start.
