# 🎮 FizzBuzz Game (Python)

Welcome to my FizzBuzz project!
This is a simple Python program that takes a number as input and applies the classic FizzBuzz logic — with a small twist.

---

## 🚀 How It Works

The program checks a number and:

* Prints **"FizzBuzz"** if the number is divisible by **3 and 7**
* Prints **"Buzz"** if the number is divisible by **7**
* Prints **"Fizz"** if the number is divisible by **3**
* Otherwise, it prints the number itself

---

## 🧠 Logic Behind It

The core of the program is a function:

```python
def fizzbuzz(n):
```

This function:

1. Takes an integer `n`
2. Evaluates divisibility conditions
3. Returns the corresponding result as a string

---

## ▶️ How to Run

1. Make sure you have Python installed
2. Run the script:

```bash
python fizzbuzz.py
```

3. Enter a number when prompted
